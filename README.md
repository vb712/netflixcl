# Marketing Site

This repository contains a static, Netflix-inspired fan site built with plain HTML, CSS, and JavaScript.

## What is included

- `index.html` for the landing page
- `player.html` for the seasonal player experience
- `style.css` for the shared styling
- `script.js` for routing, playback, and slideshow behavior
- Media folders such as `edit1`, `edit2`, `edit3`, and `edit4`

## How to run locally

Open `index.html` in a browser, or serve the folder with any static file server.

Example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploying

The project is linked to Vercel and the live site is available at `https://katya-jade.vercel.app`.

## Notes

- The site uses local media assets, so keep the folder structure intact.
- Changes pushed to the connected Git repository can be redeployed through Vercel.
